## var 使用规范

**基本原则**：默认使用 `var`，现代 IDE 已提供完善的类型提示功能。

---

### ✅ 默认使用 var

几乎所有场景都应使用 `var`：

```csharp
// 构造函数
var player = new Player();
var list = new List<int>();

// 方法返回值
var damage = CalculateDamage(attacker, target);
var data = LoadConfig(path);

// 数值类型
var distance = Vector3.Distance(a, b);
var count = 0;
var isValid = true;

// foreach 循环
foreach (var player in players) { }

// LINQ 查询
var activePlayers = players.Where(p => p.IsActive).ToList();

// 元组解构
var (success, data) = TryParse(input);
```

---

### ❌ 需显式声明类型的场景

唯一需要显式声明的情况：**需要强调接口抽象时**

```csharp
// ✅ 使用接口类型声明
IList<Player> list = new List<Player>();
IEnumerable<Player> collection = GetPlayers();
IComponent component = new TransformComponent();

// ❌ 这会掩盖接口抽象
var list = new List<Player>();
var collection = GetPlayers();
```
